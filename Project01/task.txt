REST API - JSON

GET /users - HTML Document render
GET /api/users - List all users

GET /api/users/1 - Get the user with Id 1
GET /api/users/2 - Get the user with Id 2

Dynamic Path Parameters
GET /api/users/:id
:id -> Variable

POST /api/users - Create a new user

PATCH /api/users/id - Edit the user with id 1

DELETE /api/users/1 - Delete the user with id 1

POSTMAN - Postman is a utility tool which helps in API Testing and documentation.

Middleware - Middleware are the functions which verify each and every req and 
response. We can have multiple Middleware in our function.

HTTP Headers - HTTP Headers are the meta data that are associated with the API req and res.
